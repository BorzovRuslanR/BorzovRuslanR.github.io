# Название проекта Starter
Этот проект был реализован при помощи HTML, CSS и немного JavaScript для модального окна и переключения на светлую и тёмную тему.

## Технологии
- [JavaScript](https://learn.javascript.ru/)
- [Cascading Style Sheets (CSS)](https://developer.mozilla.org/ru/docs/Web/CSS)
- [Hypertext Markup Language (HTML)](https://developer.mozilla.org/ru/docs/Learn/Getting_started_with_the_web/HTML_basics)


## Функционал приложения
- возможность смены темы
- при клике на Sing In происходит открытие модального окна
